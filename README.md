# CodeToArt - 代码可视化艺术生成器 🎨💻

## 产品简介

**CodeToArt** 是一个基于网页的艺术生成器，用户可以将 **Python 代码** 粘贴到工具中，工具会自动生成一幅抽象艺术画，使用颜色和线条表示函数调用、循环、变量关系等代码逻辑。你输入的每一段代码都会变成一幅独一无二的艺术作品！

* **展示方式**：输入一段代码（如 `print("hello")`），输出一幅梵高风格的画作。
* **支持功能**：基于代码的结构、复杂度和“情感”，生成符合其特点的艺术画。

## 使用方法 🚀

1. **访问网页**
   你可以直接访问工具页面：[CodeToArt](https://milab-bit.github.io/CodeToArt/)

2. **输入 Python 代码**
   粘贴一段 Python 代码（如 `print("Hello, World!")`），工具会自动解析代码结构并生成艺术画。

3. **生成艺术画**
   点击 **“经典算法”** 或 **“AI 注入灵魂”** 按钮，开始生成艺术作品。

4. **下载画作**
   生成的艺术画可以下载为 **PNG 图片**，保存并分享你的创作。

## 克隆项目

如果你希望将此项目克隆到本地并进行修改，可以按照以下步骤操作：

### 1. **克隆仓库**

打开终端并运行以下命令，将仓库克隆到本地：

```bash
git clone https://github.com/your-username/CodeToArt.git
```

### 2. **安装依赖**

进入项目文件夹：

```bash
cd CodeToArt
```

### 3. **运行本地服务器**

如果你希望在本地运行该工具，可以通过以下命令启动本地服务器：

```bash
npm install
npm start
```

### 4. **访问本地页面**

启动后，在浏览器中访问 `http://localhost:3000`，即可在本地使用该工具。

## 启用 AI 功能 ✨

**Gemini AI** 的 **艺术生成** 功能需要 API 密钥来启用。请按照以下步骤配置：

1. 访问 [Gemini API](https://generativelanguage.googleapis.com/) 获取 API 密钥。
2. 打开项目中的 **`index.html`** 文件。
3. 在脚本中找到 `const apiKey = ""`，将空字符串替换为你的 **Gemini API 密钥**。

   ```javascript
   const apiKey = "your-gemini-api-key";  // 替换为你自己的 API 密钥
   ```

**注意**：如果你没有设置 API 密钥，AI 功能将不可用，系统会回退到经典模式，基于常规算法生成艺术作品。

### **主要特点** 🎨

* **抽象艺术生成**：根据 Python 代码中的函数、循环、变量关系等结构，生成抽象艺术作品。
* **AI 增强功能**：通过 **Gemini AI**，分析代码的“情感”并生成符合其特征的配色方案和画作风格。
* **互动式界面**：用户可以调整抽象程度，生成不同风格的艺术作品。
* **即时反馈**：在输入代码后立即生成艺术画，并提供艺术解读。

---

### **技术实现** ⚙️

* **前端**：HTML, CSS, JavaScript（使用 **TailwindCSS** 和 **FontAwesome**）
* **AI 功能**：通过 **Gemini AI**，对输入的代码进行情感分析，生成符合代码结构的艺术风格和配色方案。
* **可下载画作**：生成的画作可以下载并保存为 **PNG** 格式。
